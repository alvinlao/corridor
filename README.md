# Corridor

A fan made version of the board game.

![screenshot](screenshot.png)

## Development

### Run

To run a local server:

```
$ npm start
```

### Build

To compile into a distributable form:

```
$ npm run-script build

dist/index.html
dist/index.xxx.js
dist/index.xxx.css
```

### Test

To run the tests:

```
$ npm test
```
